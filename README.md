# 🧠 SmartFlow — Intelligent Task Behavior System

SmartFlow is not just a task manager.

It is a system designed to analyze user behavior, automatically define task priorities, and provide intelligent feedback based on actions.

---

## 🚀 Features

- ✅ Automatic task prioritization (ALTA / MEDIA / BAIXA)
- 🧠 Behavioral analysis system
- 📊 Productivity scoring
- ⚖️ Smart balance to avoid overload
- 🔥 User level classification (FOCADO, PRODUTIVO, DESORGANIZADO)

---

## 💡 How it works

The system reads user tasks and:

- Detects priority based on keywords
- Applies rules to classify tasks
- Calculates a productivity score
- Determines the user's behavior level
- Provides feedback based on performance

---

## 💻 Technologies

- File handling
- Logic-based decision system

Most task managers only store tasks.

**SmartFlow interprets behavior.**
